# api-rest-node
## Middleware CORS
Es un mecanismo de seguridad del navegador que permite que una aplicación web acceda a recursos de un servidor en un dominio diferente al original.  En Node.js con Express, se implementa mediante el módulo cors, que actúa como middleware para agregar los encabezados HTTP necesarios (como Access-Control-Allow-Origin) a las respuestas del servidor.
